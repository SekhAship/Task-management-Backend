This repository contains the backend implementation of **Task Master**, a real-time task management system.  
The backend is built using **Node.js, Express, Socket.IO, and PostgreSQL**, and provides REST APIs along with real-time updates.

Features

- RESTful API for task management
- Real-time task updates using Socket.IO
- PostgreSQL database integration
- Task status tracking (Pending, In Progress, Completed)
- Environment-based configuration
- Deployed on Render

Tech Stack

- Node.js
- Express.js
- Socket.IO
- PostgreSQL
- pg (node-postgres)
- dotenv
- CORS
